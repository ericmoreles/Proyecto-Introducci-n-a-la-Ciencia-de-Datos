# Proyecto-Introduccion-a-la-Ciencia-de-Datos
Búsqueda de relación entre educación, nivel económico y condición social en México a nivel nacional.

Dentro de este repositorio se encuentra un análisis de la relación entre nivel económico, condición social y educación en México, usando datos obtenidos de fuentes oficiales del gobierno de México.

En la carpeta de resultados estan dos pdfs, ambos describiendo lo encontrado, pero uno en formato beamer para presentar, mientras que el otro es un reporte escrito.

En la carpeta Codigo esta el script de python que se utilizó para obtener las estadísticas y las gráficas presentadas, dentro del código se puede modificar cual(es) variable(s) se usa(n) como variable(s) predictora(s) y variable respuesta. Tambien se encuentra dentro de Codigos la carpeta Datos, aquí estan los sets de datos usados para el análisis. Se menciona en el README del codigo, pero cabe recalcar, que el script asume que los datos estan en la misma carpeta que el script a la hora de correr, sería necesario asegurarse de esto antes de correr el script, o modificar el path de las llamadas a pd.read_excel(...) dentro del codigo para tener el resultado correcto.
